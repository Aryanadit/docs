# Blog Feature

A blog built with VitePress and TinaCMS.

## Setup
```bash
npm install
```

## Development
```bash
npx tinacms dev -c "npm run dev"
```

Then open:
- Blog: http://localhost:5173/blog/
- CMS Editor: http://localhost:4001/admin/index.html

## Adding a New Post

1. Open the CMS editor
2. Click Blog Posts → Add File
3. Fill in the fields and save
4. Restart the dev server to see it in the sidebar
